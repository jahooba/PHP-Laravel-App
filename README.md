<h1>Vehicle Inspection App</h1>

This is a small full-stack web app built with PHP and Laravel that simulates a vehicle emissions inspection system.

Users can:

*    Submit vehicle inspections to database
*    View inspection results
*    View inspection list

<h2>Technical Highlights</h2>

This project demonstrates:

*    Laravel routing and controllers
*    Blade templating and layout inheritance
*    Form validation and CSRF protection
*    Business logic implementation within the controller
*    SQLite database integration
*    Feature testing using Laravel’s testing framework
*    Database assertions and validation testing
*    Separation of concerns (MVC structure)
*    Basic CSS styling for UI presentation

<h2>Testing</h2>

The application includes feature tests that verify:

*    Successful inspection creation
*    Validation error handling
*    Business logic correctness (pass/fail rules)
*    Database persistence
*    Route responses
*    Tests use Laravel’s RefreshDatabase trait to ensure isolation.

<h2>Purpose</h2>

The goal of this project was to strengthen my understanding of:

*    Laravel’s MVC architecture
*    PHP and Laravel ecosystem
*    Writing reliable automated Laravel tests
*    Structuring a maintainable PHP project
